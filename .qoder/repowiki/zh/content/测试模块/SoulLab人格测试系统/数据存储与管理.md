## 项目概述

SoulLab数据存储与管理系统是一个基于Supabase的全栈应用程序，专门为"觉醒诗社"平台提供数据存储、管理和分析功能。系统包含两个主要测试模块：SoulLab灵魂人格测试和ObjTest客体化测评，以及完整的用户认证、评论管理和后台管理系统。

该系统采用现代化的前端技术栈，结合Supabase提供的实时数据库、认证服务和对象存储，实现了高效的数据管理解决方案。系统设计注重用户体验、数据安全性和可扩展性。

## 数据安全与隐私保护

### 行级安全策略

系统采用了严格的行级安全(RLS)策略来保护用户数据：

#### 用户档案安全策略
```sql
-- 公开读取用户档案
CREATE POLICY "公开读取 profiles"
ON public.profiles FOR SELECT
USING (true);

-- 本人可更新档案
CREATE POLICY "本人更新 profiles"
ON public.profiles FOR UPDATE
USING (auth.uid() = id);

-- 本人可插入档案
CREATE POLICY "本人插入 profiles"
ON public.profiles FOR INSERT
WITH CHECK (auth.uid() = id);
```

#### 评论安全策略
```sql
-- 公开读取未隐藏评论
CREATE POLICY "公开读取未隐藏评论"
ON public.comments
FOR SELECT
USING (is_hidden = false);

-- 登录用户可发表评论
CREATE POLICY "登录用户发表评论"
ON public.comments
FOR INSERT
WITH CHECK (auth.uid() = user_id);

-- 本人可删除评论
CREATE POLICY "本人删除评论"
ON public.comments
FOR DELETE
USING (auth.uid() = user_id);
```

### 数据一致性保证

```mermaid
flowchart TD
```
